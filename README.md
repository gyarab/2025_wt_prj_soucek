# Lunaflos
Lunaflos je internetová e-shop stránka primárně určená k proeji a koupi bylinných produků. Účelem této stránky není však pouze distribuce bylin, ale zároveň také obchod se zbytkem založených produktů jako jsou například bobule, houby vonná dřeva či kořeny. Naše stránka bude na primární stránce ukazovat vybranou selekci rostlin pro rychlý přístup k primárním produktům. Na webu bude zároveň možnost procházet pomocí stránkového systému celou stránku bez přesné vědomosti ohledně produktu. Nakonec pro uživatele kteří hledají specifické rostliny či medicíny bude zavedena relativně primitivní search bar do kterého lze zadat jména specifických produktů.
